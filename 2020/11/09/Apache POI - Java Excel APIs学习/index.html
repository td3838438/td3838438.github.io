Apache POI - Java Excel APIs学习

各场提供 Java Excel 相关 API

Java Aspose Cells

Java Aspose Cells 是一种纯粹的Java授权的Excel API，开发和供应商Aspose发布。这个API的最新版本是8.1.2，发布于2014年7月，是一个丰富而厚重的API(普通Java类和AWT类的组合)设计，可以读、写和操纵电子表格Excel的组件。此API常见用途如下：

  • Excel报表，建立动态Excel报表
  • 高保真的Excel渲染和打印
  • 从Excel电子表格中导入和导出数据
  • 生成，编辑，转换和电子表格
