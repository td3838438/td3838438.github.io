CentOS7下mysql5.7忘记密码处理方法 作者：涂祎豪 停止mysql服务 service mysqld stop 修改配置文件 /etc/my.cnf 在[mysqld]下添加 skip-grant-tables 意思是忽略密码 保存并退出 启动Mysql service mysqld start 免密码登录 mysql -u root 进入数据库 use mysql 修改root密码 update user set authentication_string = password('新密码'),password_last_changed=now() where user='root'; 退出数据库 再次配置文件 /etc/my.cnf 删除 skip-grant-tables 保存退出 重启mysql服务器
