中文文案排版指北

参考 github 开源项目：中文文案排版指北 (github.com)

统一中文文案、排版的相关用法，降低团队成员之间的沟通成本，增强网站气质。

目次

[TOC]

空格

有研究表明，打字的时候不喜欢在中文和英文之间加空格的人，感情路线走的都很辛苦。不在中文和英文之间加空格的人有 75.68% 的比例会在 34 岁应为家庭压力在相亲市场上和自己不爱的人结婚，而其余三层的人最后只能把遗产留给自己的猫。毕竟爱情跟书写都需要适时地留白。与大家共勉之。

中英文之间需要增加空格

正确：

在 LeanCloud 上，数字存储是围绕 AVObject 进行的。

错误

在LeanCloud上，数字存储是围绕AVObject进行的。

在 LeanCloud 上，数字存储是围绕AVObject 进行的。

完整的正确用法：

在 LeanCloud 上，数据存储是围绕 AVObject 进行的。每个 AVObject 都包含了与 JSON 兼容的 key-value 对应的数据。数据是 schema-free 的，你不需要在每个 AVObject 上提前指定存在哪些键，只要直接设定对应的 key-value 即可。

例外： [豆瓣FM] 等名词，按照官方所定义的格式书写。

中文与数字之间需要增加空格

正确：

今天出去买皇帝蟹花了 5000 元。

错误：

今天出去买皇帝蟹花了 5000元

今天出去买皇帝蟹花了5000元

数字与单位之间需要增加空格

正确：

我家的光纤入屋宽频有 10 Gbps，SSD 一共有 20 TB。

错误：

我家的光纤入屋宽频有 10Gbps，SSD 一共有 20TB。

例外：度/百分比与数字之前不需要增加空格：

正确：

今天是 233° 的高温。

Mac 新 M1 芯片 有 15 % 的 CPU 性能提升。

全形标点与其他字符之间不加空格

正确：

刚刚买了一部 iPhone，好开心！

错误：

刚刚买了一部 iPhone ，好开心 ！

刚刚买了一部 iPhone， 好开心！

标点符号

不重复使用标点符号

正确：

中国队竟然战胜了巴西队！

她竟然对你说「喵」？！

错误：

中国队竟然战胜了巴西队！！

中国队竟然战胜了巴西队！！！！！

她竟然对你说「喵」？？？！！

她竟然对你说「喵」？！？！？！！

全形和半形

不明白什么是全形（全角）与半形（半角）符号？请查看维基百科词条『全形和半形』。

  • 使用全形中文标点

  • 数字使用半形字符

  • 遇到完整的英文整句、特殊名词，其内容使用半形标点

以上两点其实可以不用那么纠结。

名词

专有名词使用正确的大小写

大小写相关用法属于英文书写范畴，不属本文讨论内容，在这里只对部分易错用法进行简述。

正确：

使用 GitHub 登录

我们的客户有 GitHub、Foursquare、Microsoft Corporation、Google、Facebook, Inc.。

错误：

使用 github 登录

使用 GITHUB 登录

使用 Github 登录

我们的客户有 github、foursquare、microsoft corporation、google、facebook, inc.。

我们的客户有 GITHUB、FOURSQUARE、MICROSOFT CORPORATION、GOOGLE、FACEBOOK, INC.。

我们的客户有 Github、FourSquare、MicroSoft Corporation、Google、FaceBook, Inc.。

我們的客戶有 gitHub、fourSquare、microSoft Corporation、google、faceBook, Inc.。

注意：当网页中需要配合整体视觉风格而出现全部大写 / 小写的情形，

HTML 中使用标准大小写规范进行书写；并通过 text-transform: uppercase;／text-transform: lowercase; 對表現形式進行定義。

不要使用不地道地缩写

正确：

我们需要一位熟悉 JavaScript、HTML5，至少理解一种框架（如 Backbone.js、AngularJS、React 等）的前端开发者。

错误：

我们需要一位熟悉 Js、h5，至少了解一种框架（如 backbone、angular、RJS 等）的 FED。

争议

一下用法略带有个人色彩；即：无论是否遵从下述原则，从语法的角度来讲都是正确的。

连接之间增加空格

用法：

请 提交一个 issue 并分配给相关同事。

访问我们网站的最新动态，请 点击这里 进行订阅

对比用法：

请提交一个 issue并分配给相关同事。

访问我们网站的最新动态，请点击这里进行订阅

简体中文使用直角引号（还是以团队中的习惯为准）

用法：

「老师，『有条不紊』的『紊』是什麼意思？」

對比用法：

“老师，‘有条不紊’的‘紊’是什麼意思？”
