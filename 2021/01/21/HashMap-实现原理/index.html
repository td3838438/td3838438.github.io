1. HashMap 概述

HashMap 是基于哈希表的 Map 接口的非同步实现。此实现提供所有可选的映射操作，并允许使用 null 值和 null 键（最多允许一条记录的键为 null，允许多条记录的值为 null）。此类不保证映射的顺序（遍历顺序不确定），特别是它不保证该顺序恒久不变。
